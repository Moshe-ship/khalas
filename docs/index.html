Optimization
5 techniques to cut the tax.
Each technique reduces tokens differently. khalas picks the best combination for your prompt.
🏷️

Language Hint

Prepend language context to guide the tokenizer into Arabic-aware mode.

✂️

Simplify

Replace complex Arabic morphology with simpler equivalents. Same meaning, fewer fragments.

🏗️

English Backbone

Keep instructions in English, content in Arabic. Tokenizers handle the split cheaply.

🌐

Bilingual Instruction

Mix English structure with Arabic key terms. Best of both token economies.

⚡

Shorten

Remove filler words, redundancy, and verbose honorifics. Direct and cheap.

Commands
5 commands. Zero waste.

compare

Side-by-side AR vs EN token count and cost for any prompt and provider.

optimize

Rewrite Arabic prompt to use fewer tokens. Picks the best technique automatically.

savings

Show how much you save after optimization — tokens, dollars, percentage.

explain

Break down where tokens are wasted. See which words cost the most.

config

Set default provider, API keys, and preferred optimization techniques.

Get Started
Four commands to savings.
# Install $ pip install khalas # Compare AR vs EN $ khalas compare "اكتب ملخصا عن الذكاء الاصطناعي" "Write a summary about AI" --provider openai # Optimize the Arabic prompt $ khalas optimize "اكتب لي ملخصا مفصلا وشاملا عن موضوع الذكاء الاصطناعي" --provider anthropic # See your savings $ khalas savings --last # Explain where tokens are wasted $ khalas explain "اكتب لي ملخصا مفصلا وشاملا عن موضوع الذكاء الاصطناعي"
